# Security Policy
# We take security seriously. If you discover a security vulnerability,
# please report it to us as described below.

# Reporting Process
# 1. Email us at security@asklyy.tech
# 2. Include detailed steps to reproduce the issue
# 3. Do not disclose the vulnerability publicly until we've had a chance to address it
# 4. We will respond within 48 hours

# Scope
# This security policy applies to:
# - The main application at asklyy.tech
# - All subdomains
# - API endpoints
# - Third-party integrations

# Out of Scope
# - Social engineering attacks
# - Physical attacks
# - Denial of service attacks
# - Issues in third-party services we use
